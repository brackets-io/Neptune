# Neptune
##### Take your code to another world.
Neptune is a ***Drak*** theme for [Brackets](http://brackets.io/).

### Version
1.0.0

### Installation
 - Method One
  - One your brackets *Extension Manager*
  - Move to themes tab
  - Search for ***Neptune***
  
 - Method Two
  - Clone the theme to your *Extension Folder*
  - ```sh
    $ git clone 
    ```
